# ぼいぼサポーター

[VOICEVOX](https://voicevox.hiroshiba.jp/)（ずんだもん達がしゃべる無料ソフト）のキャラクターに、
**台本を書いておしゃべりしてもらう** Windows 用ツールです。

```
A『こんにちは、ミタマだよ』
B『トバリです。こんな台本を書くと…』
A『そのまま会話劇として再生されるよ！』
```

こんなテキストを再生ボタンひとつで、キャラクターの掛け合いとして聞けます。

## できること

- **台本の再生** — `A『セリフ』` の形で書いたテキストを、配役どおりの声で上から順に再生。
  音声ファイル（WAV）への書き出しもできます
- **コピーしたテキストの読み上げ** — Web の記事や AI の長い回答をコピーして、そのまま聞けます
- **AI の返事を声で聞く** — Claude Code などの AI ツールと連携して、作業しながら返事を耳で受け取れます
- **他のソフトからの読み上げ** — 自作ツールやスクリプトからテキストを送って読み上げさせられます

VOICEVOX 本体（エディタ画面）を起動しておく必要はありません。
中の音声合成エンジンだけを、必要なときに自動で立ち上げて使います。

※VOICEVOX 公式とは無関係の、非公式ツールです。

## 必要なもの

1. **Windows 10 / 11** のパソコン
2. **VOICEVOX** — [公式サイト](https://voicevox.hiroshiba.jp/)から無料でインストールできます

これだけです。ほかのソフトやプログラミングの知識は要りません。

## インストール

1. [Releases ページ](../../releases)から zip ファイルをダウンロード
2. 好きな場所に展開（解凍）する
3. 中の **`voibo-supporter.exe`** をダブルクリック

設定画面が開いて、タスクトレイ（画面右下の時計の横）にもアイコンが入ります。
次からはトレイのアイコンを右クリックすると、読み上げの ON/OFF などを操作できます。

## はじめかた

1. 設定画面の**キャスト**タブで、しゃべってほしいキャラクターを選ぶ
   （A〜H の枠にキャラクターを配役します。カードをクリックすると選択画面が開きます）
2. **台本**タブに切り替えて、こんなふうに書いてみる

   ```
   A『こんにちは！』
   B『台本のテストです』
   ```

3. **「最初から再生」**を押す — A と B に選んだキャラクターの声で再生されます

まずは「サンプル台本」ボタンで、書き方の見本をそのまま再生してみるのもおすすめです。

### 台本の書きかた（基本）

- `A『セリフ』` — A の枠のキャラクターがしゃべります
- 『』の外に書いた文は読まれません（ト書きやメモを自由に書けます）
- `Ab『セリフ』` のように小さいアルファベットを足すと、声色（スタイル）を変えられます。
  どの文字がどの声色かは、キャストタブの選択画面に書いてあります

AI に「この形式で台本を書いて」と頼んだ出力を、そのまま貼り付けて再生することもできます。
もっと細かい書き方（読みがなの指定・速さの指定など）は [リファレンス](docs/REFERENCE.md) にまとめてあります。

## AI の返事を声で聞く（Claude Code 連携）

Claude Code を使っている場合、コマンド1回で連携を設定できます:

```
voibo-supporter.exe install-hook
```

これで、AI が `A『こんにちは』` のような記法で返事を書くと自動で読み上げられます。
AI 側への伝え方は、設定画面のキャストタブにある「連携用の共有文言」をコピーして
AI に渡すだけです（詳しい説明ファイルも自動で用意されます）。

## 困ったときは

- **音が出ない** — VOICEVOX がインストールされているか確認してください。
  トレイアイコンの「読み上げ ON/OFF」が OFF になっていないかも見てみてください
- **原因がわからない** — 環境をまとめて点検するコマンドがあります（何も変更しません）:

  ```
  voibo-supporter.exe doctor
  ```

- 設定画面が VOICEVOX を見つけられないときは、画面の案内に従ってください

## 作った音声を公開するときの注意

台本から書き出した音声（WAV）を動画などで公開する場合は、
**使ったキャラクターごとの利用規約に従ってください**（多くはクレジット表記が必要です）。
各キャラクターの規約は、設定画面のキャストタブ下部にまとめて表示されます。

## 開発者向け

- 本体は Python 製で、**依存は標準ライブラリのみ**（pip 不要）。Python 3.10 以上があれば
  `python speak.py ...` でソースのまま動きます（exe と同じコマンド体系）
- コマンド一覧・記法の詳細仕様・HTTP での流し込み（serve）などは
